# Move and Solve

**Move and solve** - математическая игра, разработанная для детей 3 класса.\
Суть игры, в командах, принимая различные позы, набрать нужное количество очков, которое является ответом на пример.
Игра развивает *командную работу*, и *математическое мышление*.
## Игровой процесс
* Ученики разбиваются на несколько команд, в каждой из которых должно быть 3 человека.
* По очереди команды подходят к камере и получают пример, который необходимо решить. В примере используются такие вычислительные операции как сложение, вычитание, умножение и деление.
* Ученики, принимая различные позы, набирают очки:
  * **Прыжок** - 1 очко
  * **Наклон** - 5 очков
  * **Приседание** - 10 очков
* Дети должны набрать такое кол-во очков, которое бы являлось ответом на полученный пример.
* Далее есть несколько нюансов:
  * Если кол-во очков в какой-то момент превысило правильный ответ, то пример автоматически аннулируется и команда не получает балл.
  * После того, как команда набрала нужное кол-во очков, ответ необходимо подтвердить. Для этого все члены команды должны поднять руку.
* Если команда правильно ответила на пример, то она получает балл. Команды соревнуются между собой, тем самым повышая свою мотивацию к игре.

## Установка
0. *Рекомендуется использовать ОС Windows.*
1. Скачайте проект и откройте его в среде разработки. Используйте версию Python 3.11, чтобы код работал корректно.
2. В терминале пропишите команду `pip install -r requirements.txt` для установки всех необходимых библиотек.
3. Подключите камеру и проверьте, что ПК обнаружил ее.\
   * Если вы планируете использовать фронтальную камеру ноутбука (не рекомендуется), то необходимо заменить 25 строчку кода файла `main.py` на данную `cap = cv2.VideoCapture(0)`.
4. Расположите камеру так, чтобы в нее помещалось ровно 3 человека, которые на данный момент участвуют в игре.
5. Запустите код `файла main.py` и наслаждайтесь игровым процессом.
